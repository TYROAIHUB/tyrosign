# TYRO Sign Snap

**Tiryaki İmza Sanatçısı** — Tiryaki Agro kurumsal e-posta imza oluşturucu web uygulaması.

## Özellikler

- **İmza Sihirbazı** — Form doldur, canlı önizleme ile anında gör
- **3 Bölümlü İmza** — Logo | Dikey Çizgi | Bilgiler | Navy Sosyal Medya Bloğu
- **Otomatik Ofis Bilgileri** — Ofis seçince adres, SDN, Fax otomatik gelir
- **Title Case** — Küçük yazılan ünvanlar otomatik düzeltilir
- **Logo Yükleme** — Drag & drop veya tıkla, boyut ayarlama
- **Kopyala & İndir** — Rich HTML kopyalama (Outlook uyumlu) + HTML dosya indirme
- **TR/EN Dil Desteği** — Tek tıkla dil değiştirme
- **Dark Mode** — Koyu tema desteği
- **Mobile Responsive** — Hamburger menü, drawer sidebar
- **Outlook Uyumlu HTML** — Table-based layout, inline styles
